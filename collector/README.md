# Collector

В окне коллектора будет находиться список микроконтроллеров, которые будут подключены к системе. У каждого микроконтроллера будет свой список датчиков, которые будут подключены к нему. Датчики будут иметь имена и значения, которые будут обновляться в реальном времени.

Коллектор будет иметь возможность создать тестовый микроконтроллер с датчиками, значения которого можно будет варьировать в зависимости от выбранного датчика.

Все собранные данные коллектор отправляет в ядро ( API ). Так же отправляет события, такие как:
  - Подключение нового микроконтроллера
  - Отключение микроконтроллера
  - Обновление значения датчика

На этом работа коллектора завершается.
